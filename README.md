# felixapel's Unraid Templates

Community Applications template repository.

No installable template is currently published.

The previous CWA eBook Translate template was quarantined because it used a
moving image tag and no longer represented the project's supported security and
authentication contract. The reviewed v2.2.2 candidate image has the immutable
index digest
`ghcr.io/felixapel/cwa-ebook-translate-plugin@sha256:6fff8f7cd9210bb80439c93817edb5d813794e5e582d6c7470457f5688fb4e48`.
Its replacement XML will remain unpublished until it passes physical Unraid
acceptance and the Community Applications validation workflow.
